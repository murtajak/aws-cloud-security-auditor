# AWS Cloud Security Auditor

## Overview
This project is a serverless security auditing solution built using AWS Lambda to detect misconfigured security groups.

## Features
- Detects exposed ports (SSH, RDP, databases)
- Automated daily scans using EventBridge
- Logs results in CloudWatch
- Implements least-privilege IAM roles

## Technologies
- AWS Lambda
- Python
- CloudWatch
- EventBridge
- IAM

## Purpose
To identify security risks in cloud environments and improve overall security posture.
